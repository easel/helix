# Metrics Dashboard Generation

Create a concise metrics dashboard that turns raw data into decisions.

## Focus
- Focus on the few business, technical, and quality metrics that matter most.
- Prefer anomalies, owners, and next actions over dashboard theater.
- Avoid vanity metrics and explain why each change matters.

## Completion Criteria
- All metrics have current values and trends.
- Anomalies are explained.
- Recommendations are specific and actionable.
- Historical comparison provides context.
