# User Stories Generation Prompt
Write the smallest useful set of prioritized stories for the feature.

## Focus
- Keep stories user-centered and independently understandable.
- Split stories into vertical slices that can be tested and built independently.
- Preserve feature linkage, priority, and dependency information.
- Make acceptance criteria testable.

## Completion Criteria
- Stories map to the feature scope.
- Priorities are clear.
- Each story can drive test-first implementation.
