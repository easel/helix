# Product Vision

## Mission

Authentication-as-a-service for internal tools. We give teams a single
API-key-protected endpoint that any service can verify against, so we
stop reinventing per-service auth.

## Vision

Every internal tool authenticates the same way: send your `X-API-Key`,
get a 200 or a 401. Same revocation, same audit trail, same SLOs across
twenty backends.

## North Star

Time-to-add-a-new-internal-tool drops from days to under an hour, and
the security review is one paragraph instead of a project.
