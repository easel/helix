# temperature-converter

A tiny Node.js CLI that converts between Fahrenheit and Celsius. Used
as the example "existing project" for the helix-adopt demo — there's
already source, tests, and a README, but no HELIX artifacts yet.

## Usage

```
node src/convert.js c-to-f 100
node src/convert.js f-to-c 212
```

## Test

```
npm test
```
